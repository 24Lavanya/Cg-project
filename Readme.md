# Image Enhancement Project

This project demonstrates various image enhancement techniques using OpenCV and Python. Techniques included are:

- Histogram Equalization
- CLAHE
- Sharpening
- Smoothing (Blurring)
- Edge Detection
- Brightness and Contrast Adjustment
- Denoising
- Color Enhancement
